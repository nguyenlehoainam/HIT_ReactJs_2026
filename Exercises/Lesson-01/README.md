# 📝 Bài tập về nhà - Buổi 1

## I. Ôn tập kiến thức cũ
* Nắm vững khái niệm SPA (Single-Page Application) và MPA (Multi-Page Application).
* Khởi tạo dự án ReactJS bằng câu lệnh: `npm create vite` hoặc `yarn create vite`.
* Nắm vững tư duy và cách chia nhỏ Components.

## II. Yêu cầu tìm hiểu thêm
* Tìm hiểu công cụ quản lý package Yarn.
* Tìm hiểu về JS Modules (cách export/import).
* Phân biệt DOM, BOM và cách xử lý lỗi với `try-catch`.
* Tìm hiểu cách deploy dự án ReactJS lên nền tảng Vercel.
* Phân biệt và gọi tên các thành phần website (Tham khảo: [Video F8 Official](https://www.youtube.com/watch?v=b8Z5Cyod9oI)).

## III. Thực hành Code BTVN
* **Nhiệm vụ:** Sử dụng ReactJS code lại thành phần Header, Banner, phần "Save your time to using SoftLand" và Footer.
* **Link giao diện mẫu:** [SoftLand Theme](https://bootstrapmade.com/demo/SoftLand/)
* **Deadline:** 20h00 ngày 17/03.
* **Yêu cầu nộp:** Gửi kèm 1 link deploy (Vercel hoặc Github Page) và 1 link source code trên Github.
* **Lưu ý Git:** Bắt buộc chia nhánh (branch) trên GitHub để tránh khởi tạo lại dự án. Với mỗi BTVN là một nhánh riêng biệt.
